# Numbers & The Math Object

## Simple math with Numbers

```javascript
const num1 = 100;
const num2 = 50;
let val;
val = num1 + num2; // return 150
val = num1 * num2; // return 5000
val = num1 - num2; // return 50
val = num1 / num2; // return 2
val = num1 % num2; // return 0 ( % = Modulo )
```

## Math Object

```javascript
val = Math.PI; // return 3.141592653589793
val = Math.E; // return 2.718281828459045
val = Math.round(2.4); // return 2  → round
val = Math.ceil(2.4); // return 3 → rounded up
val = Math.floor(2.8); // return 2 → rounded to the lower unit
val = Math.sqrt(64); // return 8 → square root (racine carrée)
val = Math.abs(-3); // return 3 as an absolut
val = Math.pow(X, n); // return → Xn
val = Math.min(2, 33, 4, 1, 55, 6, 3, -2); // return -2
val = Math.max(2, 33, 4, 1, 55, 6, 3, -2); // return 55
val = Math.random(); // return a random number between 0 & 1
val = Math.floor(Math.random() * 20 + 1); // return a random number between 0 & 20 without float, rounded to the lower unit
```
