# Dates & Times

```javascript
//Jour de création → Mardi 12 février 2019
let val;
const today = new Date(); //return Tue Feb 12 2019 12:51:31 GMT+0100 (Central European Standard Time) {} as an object
let birthday = new Date('9-10-1981 11:25:00');
birthday = new Date('September 10 1981');
birthday = new Date('9/10/1981');

val = today.getMonth(); // return 1 → [0: Janvier, 1: Février, 2: Mars, …] as a Number
val = today.getDate(); // return 12 as a Number
val = today.getDay(); // return 2 → a l’américaine : [0 : Dimanche, 1: Lundi, 2: Mardi, …] as a Number
val = today.getFullYear(); // return 2019 as a Number
val = today.getHours(); // return 12 as a Number
val = today.getMinutes(); // return 51 as a Number
val = today.getSeconds(); // return 31 as a Number
val = today.getMilliseconds(); // return 217 as a Number
val = today.getTime(); // return Tue Feb 12 2019 17:56:34 GMT+0100 (Central European Standard Time) {}

//date test : Thu Sep 10 1981 00:00:00 GMT+0200 (Central European Summer Time) {}
→ as an Object
birthday.setMonth(2); → →  // return Thu Mar 10 1981 00:00:00 ...
birthday.setDate(12); → →  // return Thu Mar 12 1981 00:00:00 ...
birthday.setFullYear(1985);// return Thu Mar 12 1985 00:00:00 ...
birthday.setHours(3); → →  // return Thu Mar 12 1985 03:00:00 ...
birthday.setMinutes(30);→  // return Thu Mar 12 1985 03:30:00 ...
birthday.setSeconds(25);→  // return Thu Mar 12 1985 03:30:25 ...
```
